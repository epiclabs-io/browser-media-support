Media Source Extensions (MSE) is a Javascript API that let's you build video players that support streaming protocols beyond progressive media download of MP4/WebM files. It is one of the core pieces of modern HTML5 video players which support HTTP chunk based streaming protocols like MPEG DASH ( dash.js) or HLS (hls.js), that allows you to implement advanced streaming features like:
  • Adaptive Streaming. Streaming quality automatically adapted attending to evolution of different metrics (ex: network conditions, frame drops, etc)
  • Live Streaming with time shift (DVR)
  • Ad insertion feature in which ad chunks are inserted between content chunks

Together with MSE, Encrypted Media Extensions (EME) is a Javascript API that allows you to use DRM capabilities of your browser. With both APIs you can build HTML5 video players which support streaming protocols and features that before MSE/EME were only possible with the help of browser plugins like Flash or Silverlight.
